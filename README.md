# k8s-php-nginx-ingress-pv

Learning kubernetes by creating deployment, service, persistent volume. In this case I'm using nginx as ingress, and metallb for LoadBalancing service.

## Installation

- [VirtualBox](https://wiki.archlinux.org/index.php/VirtualBox)
- [docker](https://docs.docker.com/engine/install/ubuntu/)
- [minikube](https://kubernetes.io/docs/setup/learning-environment/minikube/)
- [nginx-ingress](https://kubernetes.github.io/ingress-nginx/)
- [metallb](https://metallb.universe.tf/)
- [bitnami/php-fpm](https://hub.docker.com/r/bitnami/php-fpm/)
- [nginx](https://hub.docker.com/_/nginx)
